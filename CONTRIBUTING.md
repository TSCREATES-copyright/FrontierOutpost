# 🤝 Contributing to Frontier Outpost

Thanks for your interest in contributing!

## 📋 Guidelines

- Keep changes focused and minimal
- Follow existing project structure
- Do NOT introduce frameworks or build tools
- Maintain modular ES6 structure

## 🚀 Workflow

1. Fork the repo
2. Create a branch:
   feature/your-feature-name
3. Make your changes
4. Commit clearly:
   feat: add new building mechanic
5. Open a Pull Request

## 🧠 Code Standards

- Clear module boundaries
- No global pollution
- Keep systems decoupled
- Maintain performance

## 🎮 UI Contributions

- Must not block gameplay visibility
- Must be responsive
- Must remain lightweight

## 🐞 Bugs

- Provide steps to reproduce
- Include screenshots if possible

Thanks for helping build Frontier Outpost!
